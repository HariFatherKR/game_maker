# IdleFarm Roguelike

방치형 농사 게임 + 로그라이트 증강체 시스템을 결합한 Steam/모바일 멀티플랫폼 게임

## 게임 소개

황폐한 농장을 되살리고, 신들의 축복을 받아 최강의 농부가 되세요!

### 핵심 특징

- **방치형 농사**: 작물을 심고 키우며, 오프라인에서도 성장
- **로그라이트 증강체**: 매 런마다 새로운 증강체 조합으로 플레이
- **신 시스템**: 10명의 신에게 봉헌하고 강력한 시너지 효과 획득
- **시즌 시스템**: 봄/여름/가을/겨울 4계절을 거치며 진행
- **위협 시스템**: 해충과 재해에 대응하며 농장을 지키세요
- **펫 시스템**: 귀여운 펫들이 농장 운영을 도와줍니다
- **6개 바이옴**: 다양한 환경에서 특수 작물 재배
- **5개 엔딩**: 당신의 선택에 따른 다양한 결말
- **하드모드**: 진정한 도전을 원하는 농부를 위한 극한 난이도

## 기술 스택

| 구성요소 | 기술 |
|----------|------|
| 게임 엔진 | Godot 4.5+ (GDScript) |
| Steam 연동 | GodotSteam |
| 모바일 | React Native + @borndotcom/react-native-godot |
| 저장 | AES-256-CBC 암호화 |

## 프로젝트 구조

```
game_maker/
├── godot/                    # Godot 핵심 게임
│   ├── scripts/
│   │   ├── autoload/         # 핵심 싱글톤 (7개)
│   │   ├── core/             # 데이터 클래스
│   │   ├── farm/             # 농사 시스템
│   │   ├── roguelike/        # 증강체 시스템
│   │   ├── threats/          # 위협 시스템
│   │   ├── pets/             # 펫 시스템
│   │   ├── story/            # 스토리/튜토리얼
│   │   ├── social/           # 소셜 기능 (4개)
│   │   ├── endgame/          # 엔드게임 콘텐츠 (3개)
│   │   ├── platform/         # 플랫폼별 코드
│   │   └── ui/               # UI 컴포넌트
│   └── scenes/               # 씬 파일
├── mobile/                   # React Native 모바일 앱
├── shared/                   # 공유 데이터
└── scripts/                  # 빌드 스크립트
```

## 빌드 명령어

### Godot 개발
```bash
# 에디터 실행
cd godot && godot --path . --editor

# 디버그 실행
cd godot && godot --path . --debug
```

### Steam 빌드
```bash
./scripts/build_steam.sh --debug
./scripts/build_steam.sh --release
```

### Mobile 빌드
```bash
cd mobile && pnpm install && pnpm ios
cd mobile && pnpm install && pnpm android
```
